

# Basic PHP Exercises (Personal Practice)

This repository contains basic PHP content created as personal practice. It includes simple examples of syntax and fundamental language concepts.

## Contents
- Printing to screen (echo)
- Comments (single-line and multi-line)
- Variables and data types
- Arithmetic operations
- Booleans and ternary operators
- Constants (define and const)
- Arrays and associative arrays (Dictionaries)
- Loops
- Classes and objects
- Regular expressions (preg_match)


## Usage
You can run the `index.php` file on a local server or using the terminal with:

```bash
php index.php
```
